
# soNative
implementation of native components using weex and VueJS 2

## Getting started
- download
- `cd` into repo
- npm install
- `cd` into `platforms/ios`
- `open .`
- open `WeexDemo.xcworkspace` on xCode
- build to run on your connected ios device
